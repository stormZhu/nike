## 行高类

### leading-normal

**类型**：行高控制
**定义**：设置元素的行高为正常大小
**值**：`line-height: 1.5;`
**用途**：

- 为大多数文本提供平衡的可读性与空间效率
- 适合正文、导航链接等需要良好可读性的文本元素
  **示例**：

```jsx
// 导航链接使用正常行高
<a className="leading-normal text-lg">Home</a>
```

**在项目中的应用**：

- 文件：`src/components/Nav.jsx` 第17行
- 用于导航菜单链接，确保大字体文本仍有良好的可读性
